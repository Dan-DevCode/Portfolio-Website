OG IMAGE (og.png)
=================

Please create a 1200x630px image for social media previews (Open Graph).
This image will be used when your portfolio is shared on social media platforms.

Recommended:
- Use a design tool (Figma, Canva, Photoshop, etc.)
- Include your name and a tagline or visual element
- Ensure text is readable at small sizes
- Save as PNG format
- Place the file here as: og.png

Example dimensions: 1200px × 630px
Format: PNG (or JPEG if needed)
